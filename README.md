## Highlights

- Supports both simplified and traditional characters
- Powered by the [CC-CEDICT](https://cc-cedict.org/wiki) Chinese-English dictionary
- Displays Hanyu Pinyin with tone colors for faster memorization
- Toggle on/off with a single click
- Highlights the characters currently being translated
- Keyboard navigation to move word by word or character by character
- Built-in word list — save words and export to [Anki](https://apps.ankiweb.net)
- Links to grammar notes on the [Chinese Grammar Wiki](https://resources.allsetlearning.com/chinese/grammar)
- [Skritter](https://skritter.com) integration for vocabulary export
- Optional Zhuyin ([Bopomofo](https://en.wikipedia.org/wiki/Bopomofo)) display


## How It Works

Click the PinyinPop icon in your browser toolbar to enable the extension — the icon shows **On** when active. Then hover over any Chinese text on a webpage. A pop-up appears showing:

- Chinese characters (simplified and/or traditional)
- Pinyin with tone colors
- English definition(s)

## Tone Colors

By default:

| Tone | Color  |
|------|--------|
| 1st  | Red    |
| 2nd  | Orange |
| 3rd  | Green  |
| 4th  | Blue   |

Three color schemes (Standard, Pleco, Hanping) are available in options.


## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| <kbd>N</kbd> | Next word |
| <kbd>B</kbd> | Previous character |
| <kbd>M</kbd> | Next character |
| <kbd>R</kbd> | Save word to word list |
| <kbd>C</kbd> | Copy entry to clipboard |
| <kbd>S</kbd> | Open in Skritter |
| <kbd>G</kbd> | Open Chinese Grammar Wiki entry |
| <kbd>T</kbd> | Look up on Tatoeba |
| <kbd>A</kbd> | Move pop-up position |
| <kbd>X</kbd> / <kbd>Y</kbd> | Move pop-up up / down |
| <kbd>Esc</kbd> | Close pop-up |
| <kbd>Alt</kbd>+<kbd>W</kbd> | Open word list |
| <kbd>Alt</kbd>+<kbd>1</kbd> | LINE Dict |
| <kbd>Alt</kbd>+<kbd>2</kbd> | Forvo (pronunciation) |
| <kbd>Alt</kbd>+<kbd>3</kbd> | Dict.cn |
| <kbd>Alt</kbd>+<kbd>4</kbd> | iCIBA |
| <kbd>Alt</kbd>+<kbd>5</kbd> | MDBG |
| <kbd>Alt</kbd>+<kbd>6</kbd> | Reverso |
| <kbd>Alt</kbd>+<kbd>7</kbd> | MoeDict |


## Word List

Press <kbd>R</kbd> while a pop-up is open to save the entry. Open the word list with <kbd>Alt</kbd>+<kbd>W</kbd>. From there you can export to a text file or import into Anki.


## Options

Right-click the PinyinPop icon → **Options** to configure:

- Simplified vs. traditional display
- Tone color scheme
- Font size
- Zhuyin (Bopomofo) display
- Skritter region
